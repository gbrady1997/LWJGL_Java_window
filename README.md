# LWJGL_Java_window
Simple Window Creation Using LWJGL In Java
compatable on Imac With Intel Processor
to compile and run in the project directory in The UNIX MacOs Terminal:

javac -cp "lib/*" -d out src/Main.java && java -XstartOnFirstThread -cp "out:lib/*" Main

And A Simple Blank GUI Window Should Appear.
--This GUI might not have Apple Silicon CPU Compability (yet).
